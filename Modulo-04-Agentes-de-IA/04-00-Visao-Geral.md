# [M04] Agentes de IA — Visão Geral

## O que este módulo resolve

O que é um agente, afinal? Como um LLM que só "prediz o próximo token" consegue executar tarefas complexas, usar ferramentas e tomar decisões? Este módulo cobre a arquitetura de agentes, o padrão ReAct, LangChain e memória.
