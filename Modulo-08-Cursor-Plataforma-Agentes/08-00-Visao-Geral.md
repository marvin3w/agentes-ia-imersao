# [M08] Cursor como Plataforma de Agentes — Visão Geral

## O que este módulo resolve

Cursor não é só um editor — é uma plataforma de agentes com primitivas próprias: Rules, Skills, Hooks, MCP servers, CLI, SDK. Este módulo documenta como usar o Cursor como ambiente de automação inteligente, com foco no contexto do vault Marcus-Docs.
