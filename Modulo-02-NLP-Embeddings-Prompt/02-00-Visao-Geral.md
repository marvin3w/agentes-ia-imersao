# [M02] NLP, Embeddings e Engenharia de Prompt — Visão Geral

## O que este módulo resolve

M01 explicou como o modelo funciona internamente. M02 explica como trabalhar com linguagem de forma eficaz: o que são embeddings e por que são a fundação do RAG, como formatar prompts que extraem o máximo do modelo, e quais são os padrões avançados que separam prompts mediocres de prompts que realmente funcionam.

## Por que este módulo importa especialmente para você

Se você já usa LLMs há tempo, provavelmente tem intuição sobre prompt engineering mas não tem o vocabulário formal. Este módulo formaliza o que você já sabe e adiciona o que provavelmente ainda não sabe — especialmente embeddings, que são a base técnica do RAG (M03).
