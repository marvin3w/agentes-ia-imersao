# [M07] MCP e Protocolos Modernos — Visão Geral

## O que este módulo resolve

Model Context Protocol (MCP) é o padrão que está unificando como agentes se conectam a ferramentas. Este módulo cobre do conceito à criação de servidores MCP customizados — com foco no ecossistema que o operador já usa no vault.
