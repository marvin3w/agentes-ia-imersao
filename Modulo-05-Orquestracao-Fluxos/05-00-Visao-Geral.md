# [M05] Orquestração e Fluxos com LangGraph — Visão Geral

## O que este módulo resolve

Agentes simples (M04) funcionam para tarefas lineares. Tarefas complexas com ramificação, estado persistente e múltiplos agentes precisam de orquestração. LangGraph é o framework padrão para isso.
