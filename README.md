# Imersão em Agentes de IA — Do Conceito ao Agente Operacional

> **Para quem:** praticante com experiência em IA/LLMs que quer solidificar os fundamentos conceituais e dominar o stack moderno de agentes (RAG, LangGraph, MCP, multi-agentes).
>
> **Espírito:** CS50 (rigor conceitual, "por quê" antes do "como") aplicado ao contexto de 2025-2026, com foco em quem já usa ferramentas mas quer entender o que está acontecendo por baixo.
>
> **Referências de ementa:** CS50 AI (Harvard) + Formação Agentes IA (Alura) + Introduction to Neural Networks (Harvard) + contexto vault-specific (Cursor/MCP).

## Progressão de Aprendizagem

```
[M01] Fundamentos       → Por que ML funciona? O que são redes neurais? O que é um Transformer?
  ↓
[M02] NLP e Prompt      → Como o modelo "lê" texto? O que são embeddings? Como criar bons prompts?
  ↓
[M03] RAG               → Como dar memória ao modelo? O que é um banco vetorial? Como montar um pipeline RAG?
  ↓
[M04] Agentes           → O que é um agente? Como ele usa ferramentas? O que é ReAct?
  ↓
[M05] Orquestração      → Como controlar fluxos complexos? O que é LangGraph? Como modelar estados?
  ↓
[M06] Multi-Agentes     → Como agentes cooperam? O que é supervisor/worker? Como debugar?
  ↓
[M07] MCP               → O que é MCP? Como criar servidores? O que é Backend for Agents?
  ↓
[M08] Cursor            → Como usar o Cursor como plataforma de agentes? Rules, Skills, CLI, SDK.
```

## Como usar este material

1. **Leitura progressiva:** cada módulo pressupõe os anteriores. Não pule módulos sem ler ao menos o `00-Visao-Geral.md` de cada um.
2. **Auto-reflexão:** todo documento tem uma seção de perguntas para internalização. Responda mentalmente antes de avançar.
3. **Conexões:** links entre documentos formam uma rede — explore os `## Conexões` de cada doc.
4. **Recursos externos:** cada documento aponta para materiais de referência (CS50, Alura, Harvard, documentação oficial).
