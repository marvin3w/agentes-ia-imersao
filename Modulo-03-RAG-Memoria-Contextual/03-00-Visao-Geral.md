# [M03] RAG e Memória Contextual — Visão Geral

## O que este módulo resolve

Como dar ao modelo acesso a informações que não estavam no treinamento, de forma confiável e verificável? RAG (Retrieval-Augmented Generation) é a resposta padrão da indústria. Este módulo cobre do conceito à implementação.
